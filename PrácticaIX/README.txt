En esta carpeta .zip, llamada PracticaIX, se encuentran los siguientes archivos:

	1. EnunciadoPractica.pdf: Contiene el enunciado con lo que se pide en la práctica.
	2. CreacionMatrices.cpp: Contiene el código necesario para crear matrices y meterlas en un fichero
	de texto, pasando como argumento el número de filas que quieres que tenga dicha matriz.
	3. CodigoPracticaVIII.cpp: Contiene el código de la práctica en el lenguaje de programación C++.
	En él se pueden encontrar las funciones de comprobación de que la matriz solución X es correcta,
	la de comprobación de si la matriz de coeficientes A es tridiagonal y la de resolución del sistema
	de ecuaciones por el método LU para sistemas tridiagonales. Además, se comprueba el tiempo de ejecución.
	Los datos de las matrices se leen de ficheros externos creados previamente (CreacionMatrices.cpp).
	4. InformePracticaIX.PabloGradolphOliva.pdf: Contiene el informe de la práctica en el que se 
	comentan los resultados obtenidos durante la misma en formato pdf.
	5. matrizA.txt y matrizF.txt: Son los ficheros dónde se guardan las matrices creadas en CreacionMatrices.cpp
	Estás matrices son las que se leeran posteriormente en el código principal.
	6. ResultadoLuTridiagonal.txt: Tras ejecutar el código principal de la práctica, el resultado de la matriz
	solución se guardará en este fichero.
