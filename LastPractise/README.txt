En la carpeta PracticaXIX.zip encontramos los siguientes archivos:

	1. CodigoPracticaXIX.cpp: Se desarrolla el código necesario para la elaboración de la práctica en lenguaje de programación C++.
	2. InformePractica.pdf: Informe en el que se discuten y comentan los resultados obtenidos durante la práctica.
	3. Pt19_Plata.txt: Resultados obtenidos a partir del programa para el caso del material plata.
	4. Pt19_Oro.txt: Resultados obtenidos a partir del programa para el caso del material oro.
	5. Pt19_Aluminio.txt: Resultados obtenidos a partir del programa para el caso del material aluminio.
	6. Pt19_HierroFundido.txt: Resultados obtenidos a partir del programa para el caso del material hierro fundido.
